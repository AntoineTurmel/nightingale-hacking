Detailed Description

[DEPRECATED]

The backscanner can be started via the start() method and stopped via stop(). Local media and local area network shares benefit from a threaded backscanning process for extra speed.

To qualify as local media the URI for the piece of media must begin with the following scheme:

  • file://.

Remote media is scanned from the main thread using timer based workers.

The metadata backscanner is a service and should be called via do_GetService.
